# wyrd-automation-n8n

This repository contains n8n workflow exports for automation tasks.

## Workflows

- `Lead Follow-up System (Corrected).json`
  - Lead follow-up automation flow.
- `Weekly CRM Report.json`
  - Weekly CRM reporting automation flow.

## How to use

1. Open your n8n instance.
2. Go to **Workflows** -> **Import from File**.
3. Select one of the `.json` files from this repository.
4. Review credentials, webhook URLs, and node settings.
5. Save and activate the workflow.

## Notes

- These files are exported workflow definitions.
- Update environment-specific settings before running in production.
